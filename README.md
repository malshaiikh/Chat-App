# Chat App

FlutterChatX is a minimal yet modern chat app built with **Flutter** and **Firebase**. It offers a clean user interface and real-time messaging, making it perfect as a starter project for Flutter developers.

## Features

- **User Authentication**: Sign up, login, and logout.
- **Chat Rooms**: Send and receive messages in real-time with all users.
- **Dark Mode**: Switch between light and dark themes via the sidebar settings.
- **Smooth Animations**: Scroll animations enhance the chat experience.
- **Responsive UI**: Works seamlessly on mobile devices.

## Technologies Used
Flutter
Firebase Auth
Firebase Firestore
